# QuizApp

Проектът представлява
интерактивна платформа за
решаване на тестове, която
позволява на потребителите да
тренират знанията си чрез онлайн
тестове.

## Технологичен Стек

PHP, HTML, CSS, MySQL, Apache,
phpMyAdmin

## Основни Функции (MVP)

Регистрация на потребители
Вход в системата
Избор на категория
Зареждане на въпроси
Изчисляване и показване на
резултати
Структура на базата данни
Изход от системата
