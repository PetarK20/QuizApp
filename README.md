# QuizApp

Проектът представлява
интерактивна платформа за
решаване на тестове, която
позволява на потребителите да
тренират знанията си чрез онлайн
тестове.

## Технологичен Стек

PHP, HTML, CSS, MySQL, Apache,
phpMyAdmin

## Основни Функции (MVP)

1. Регистрация на потребители
2. Вход в системата
3. Избор на категория
4. Зареждане на въпроси
5. Изчисляване и показване на
6. резултати
7. Структура на базата данни
8. Изход от системата

## Потребители

Ученици, студенти и
преподаватели, както и всеки,
който желае да проверява
знанията си онлайн.
